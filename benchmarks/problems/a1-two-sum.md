# A1: Two Sum

## Problem

Given an array of integers and a target sum, return the indices of two numbers that add up to the target.

## Input

- `nums`: array/list of integers
- `target`: integer

## Output

- Tuple/pair of two indices (0-based)
- Exactly one solution exists

## Constraints

- 2 ≤ nums.length ≤ 10⁴
- O(n) solution expected (use a hash map)

## What This Tests

- Hash map creation and lookup
- Iteration with index tracking
- Returning a compound value (tuple/struct/array)
- Optional/nullable handling (key might not exist in map)
